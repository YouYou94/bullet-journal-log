# Movie Picker

Rest API를 이용하여 TMDB 영화 데이터를 받아와 검색

</br>
</br>

## 1. 개발 목표

- 타입스크립트 및 Rest API를 이용하여 영화 검색 서비스 프론트엔드 구현

</br>
</br>

## 2. 사용 기술 및 배포

### 2.1 사용 기술

- TypeScript
- React
- Styled-Components
- Axios
- Recoil
- Lodash

### 2.2 배포
